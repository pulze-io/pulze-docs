RenderFlow Cloud uses a credit-based system. You buy credits, submit jobs, and credits are deducted automatically. For current credit packages and pricing, visit [pulze.io](https://pulze.io/pricing).

## How credits work

Credits are consumed per node-hour. The credit cost covers everything: render engine licenses, plugin licenses, asset uploads, rendered frame downloads, and cloud storage. There are no additional charges on top of credits.

## No queues, no priority tiers

Every customer gets the same treatment. There is no system where paying more gives you resources sooner. When you submit a job, a node is assigned to it as soon as one is available.

## Team credits

Studios can maintain a shared credit pool that anyone on the team can use. You can also create separate teams with allocated credits for different projects or departments. When submitting a cloud job, the artist selects which team's credits to use.
