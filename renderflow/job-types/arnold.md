RenderFlow renders Arnold scene files (`.ass`) directly with the `kick` command-line renderer, no host application needed on the node. Export an `.ass` from 3ds Max, Maya, Houdini, Cinema 4D, or any Arnold-capable app, and the farm renders it.

## How to submit

In the RenderFlow app, go to Jobs > Create and select the **Arnold** template, then browse for a .ass file. Set the resolution and frame range, and submit.
