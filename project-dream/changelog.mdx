**New Features:**

- New Corona and V-Ray VFB integration (3ds Max only)

**Updates:**
- Expanded support for latest software versions: 3ds Max 2026, Revit 2025/2026, Blender 4.5, Maya 2026, SketchUp 2025, and Cinema 4D 2025/2026

**Fixes:**
- Resolved Cinema 4D viewport capture issues
</Update>

<Update label="0.7.0" description="July 9, 2024">
Project Dream 0.7.0 introduces Style Transfer, mask painting, a reworked Generate pipeline, and major Creative Upscale improvements.

**New Features:**
- New Style Transfer feature - apply the style of one image to another
- New mask painting on input images to apply effects only to specific areas
- New Restoration slider in Creative Upscale to directly control the amount of added detail
- Added job-type-related prompt placeholders
- Added plugins for Blender 4.1, SketchUp 2024, and Maya 2025

**Updates:**
- Creative Upscaling is now even faster and better, producing results up to 8K
- Reworked the algorithm for all Generate jobs and consolidated down to a single input image method
- Reference images in Generate jobs now work the same way as in Style Transfer
- Prompt-only generations are 1.5x larger and slightly faster
- Removed Variations jobs - result images now load into the Generation input instead
- Reworked the Character Enhancer for more realistic results and slightly faster output
- Each feature can now use its own input image for better and faster iteration
- Enhance Character and Creative Upscale options now load generated images into their respective input fields
- Moved the Magic Salt toggle to the Prompt input box
- Added missing continuous workflow buttons to result views
- Minor UI updates
</Update>

<Update label="0.6.0" description="April 3, 2024">
Project Dream 0.6.0 expands plugin support across major DCCs, introduces the VividDreams model, and adds Creative Upscale.

**New Features:**
- Added new plugins for Revit, Rhino, SketchUp, Maya, Cinema 4D, and Blender
- Added support for 3ds Max 2025
- Added a brand new model called VividDreams
- Added a new Image Strength slider alongside Creativity for finer control over input image influence
- Added new Creative Upscale job type
- Added a new, more extensive model browser

**Updates:**
- Most input methods and models have been renamed to better reflect their usage and avoid confusion
- Character Detailer can now be used on almost any job type
- Small UI updates

**Fixes:**
- Many small bugfixes
</Update>
