Scene Manager is Pulze's tool for organizing and managing complex scenes in 3ds Max. It helps you keep your projects structured and your workflow efficient by grouping all possible settings into modular, customizable panels.

<Frame caption="Scene Manager overview">
  <img src="/images/scene-manager/sm_modules.png" alt="Scene Manager overview showing setup list, modules, and per-setup controls" />
</Frame>

Every setting in Scene Manager is tied to a **setup**. A setup is a snapshot of your scene configuration - which camera to use, what resolution, which lights are on, what render settings to apply. You create as many setups as you need and switch between them instantly. See [Managing Setups](/scene-manager/setups) for details.

The modules are organized around natural workflow stages. Start with [Camera & Framing](/scene-manager/camera-and-framing) to define your shots, set up your [Lighting & Environment](/scene-manager/lighting), control what's visible in the scene with [Scene Control](/scene-manager/scene-control), and configure your [Render Settings](/scene-manager/rendering). For advanced automation, the [Scripting](/scene-manager/scripting) modules let you run custom MaxScript at various points in the process.

When you're ready to render, you can [Bake Setups](/scene-manager/bake-setups) to native scene states for batch rendering, or submit everything directly to [RenderFlow](/renderflow/job-types/scene-manager) with a single click.

## Prerequisites

- Scene Manager requires a constant internet connection.

## Supported software

- **3ds Max** 2019 - 2026
- **V-Ray 7**, **Corona**, **FStorm**, **Arnold**, **Redshift**

<Info>
All other 3rd party and built-in render engines for 3ds Max will also work with limited functionality.
</Info>

## Installation

Please close all running 3ds Max instances. Run the downloaded exe and follow the installer's instructions.

<Frame caption="Scene Manager installer">
  <img src="/images/scene-manager/sm_installer.png" alt="Scene Manager installer" />
</Frame>

After the installation is done, start 3ds Max and go to **Customize > Customize User Interface > Toolbars** and choose the Category **Pulze** to create a toolbar button.

<Frame caption="Creating a toolbar button in 3ds Max">
  <img src="/images/scene-manager/sm_max_toolbar.png" alt="3ds Max toolbar setup" />
</Frame>

The installer also supports silent installation by using the `--silent` parameter. This option will install Scene Manager for all versions of 3ds Max that are available on the computer.

## Next steps

Once installed, start by [creating your first setups](/scene-manager/setups). From there, configure your [cameras and framing](/scene-manager/camera-and-framing), set up [lighting](/scene-manager/lighting), and define your [render settings](/scene-manager/rendering).
