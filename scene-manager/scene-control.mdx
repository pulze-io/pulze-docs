title: "Scene Control"

---

These modules control what's visible in each [setup](/scene-manager/setups). Whether you need to toggle entire layers, individual objects, external references, or product variants, Scene Manager lets you define the exact scene composition for each shot.

## Layers

For each setup you can add multiple layers from your scene and control their visibility. There are multiple ways to add your layers: you can search or select them from the dropdown, or by selecting the layers in max you can load them in by pressing the fetch button.

</Frame>

## Objects

With the Objects module you can add your selected geometry from max and control the visibility per setup. It's an alternative to the Layers module that does almost the same, but there are cases when the usage of this is much easier.

<Frame caption="Objects module with per-object visibility, transform, and material controls">
  <img src="/images/scene-manager/sm_module_objects.png" alt="Objects module with per-object visibility, transform, and material controls" />
</Frame>

<Warning>
It is not recommended to add more than 10-20 objects to this module. If you would like to control a bigger portion of the scene, use the Layers module instead.
</Warning>

## Xref Scene

The Xref Scene module will let you select files that are loaded into your scene. You can enable/disable and set the visibility of these files per setup.

<Frame caption="Xref Scene module enabling and disabling external references per setup">
  <img src="/images/scene-manager/sm_module_xref_scene.png" alt="Xref Scene module enabling and disabling external references per setup" />
</Frame>

## Variants

You can define a list of layers where each item will represent a version of a product. This way you can easily create and render hundreds of variations within one state. Perfect solution if you are doing product renderings.

<Frame caption="Variants module with a list of product variations defined by layers">
  <img src="/images/scene-manager/sm_module_variants.png" alt="Variants module with a list of product variations defined by layers" />
</Frame>

When combined with the [Render Output](/scene-manager/rendering#render-output) module's path rules, each variant can automatically get its own output filename and folder.

## Next steps

With your scene composition defined, set up your [Render Settings](/scene-manager/rendering) to configure output paths, render settings, and render elements per setup.
