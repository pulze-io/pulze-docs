## Features

Scene Manager provides a comprehensive set of tools for managing your scenes.

<AccordionGroup>
  <Accordion title="Scene organization">
    Organize scenes into groups, folders, and hierarchies to keep your projects tidy.
  </Accordion>
  <Accordion title="Quick navigation">
    Jump between scenes instantly with search and bookmarks.
  </Accordion>
  <Accordion title="Batch operations">
    Apply changes across multiple scenes at once to save time.
  </Accordion>
  <Accordion title="Customizable workspace">
    Tailor the Scene Manager interface to fit your workflow.
  </Accordion>
</AccordionGroup>
