The Jobs page is your central hub for monitoring and controlling all render activity. Here you can track progress, manage priorities, and take action on your jobs.

## Job list

The job list shows all active jobs with key information at a glance:

- **Name** — Job title and software icon
- **Status** — Current state (Working, Pending, Completed, etc.)
- **Progress** — Visual bar showing completed, working, and pending tasks
- **Tasks** — Count of completed vs total tasks
- **User** — Who submitted the job
- **Priority** — Job importance (1-100)
- **Pool** — Which node group is processing the job

The footer shows task counts across all jobs: Total, Completed, Working, Pending, and Suspended.

## Filtering jobs

Use filters to find specific jobs:

- **Status** — Show only Working, Pending, Completed, Suspended, etc.
- **My Jobs** — Show only jobs you submitted
- **Pool** — Filter by assigned pool (when multiple pools exist)

## Job actions

Right-click a job or use the action menu to control it:

| Action | Description |
|--------|-------------|
| **Start** | Begin processing a suspended or scheduled job |
| **Stop** | Pause the job immediately |
| **Finish & Stop** | Complete current tasks, then pause |
| **Reset** | Reset all tasks back to pending |
| **Edit** | Modify job settings |
| **Copy** | Duplicate the job for resubmission |
| **Archive** | Move to archive storage |
| **Delete** | Permanently remove the job |

Some actions are only available for certain job states. For example, you can only delete jobs that are Completed or Suspended.

## Quick edit

For fast adjustments without opening the full editor, use **Quick Edit** from the context menu:

- **Whitelist nodes** — Restrict to specific nodes
- **Node limit** — Set maximum concurrent nodes
- **Priority** — Adjust urgency

Select multiple jobs to apply changes in bulk.

## Job details

Click a job to open the details panel showing:

### Statistics

- Overall progress percentage
- Total render time
- Remaining time estimate
- Estimated completion date
- Average frame time

### Tasks

View all tasks with their status, progress, assigned node, and timing. Switch between:

- **List View** — Detailed table with sorting and pagination
- **Grid View** — Visual grid with color-coded status

<Frame caption="Task list in job details">
  <img src="https://placehold.co/900x400?text=Task+List" alt="Task list" />
</Frame>

Use the controls above the task list to:

- **Jump to working task** — Scroll directly to the currently rendering task
- **Select frame list** — Enter a range or list of frames (e.g., `1,3,5-12`) to select specific tasks
- **Filter** — Show only tasks with certain statuses

### Task actions

Right-click tasks for additional control:

| Action | Description |
|--------|-------------|
| **Reset** | Return completed or disabled tasks to pending status |
| **Select By → Node** | Select all tasks rendered by the same node(s) |
| **Select By → Status** | Select all tasks with the same status |
| **Disable** | Prevent tasks from being processed |
| **Enable** | Re-enable disabled tasks |
| **Complete** | Mark tasks as finished without rendering |

### Nodes

See which nodes are working on this job, have completed tasks, or encountered errors.

### Scene properties

View scene configuration including file path, resolution, camera, frame range, and output paths.

### Job properties

View job settings including priority, user, pool, start/finish times, schedule date, and cleanup settings.

### Requirements

See the software requirements the job needs — which applications and versions are required to render.

### Dependencies

If the job depends on other jobs, see their status here (Waiting or Resolved).

### Notifications

View configured notifications for job completion or failure (email, webhook).

## Errors

When tasks fail, an error indicator appears on the job. Click it to view error details and logs for each node that encountered problems.

## Archive

Completed jobs can be archived to keep your job list clean. Access archived jobs from the **Archive** button to:

- Search archived jobs
- Restore jobs back to active status
- Permanently delete archived jobs

## Bulk operations

Select multiple jobs to perform actions on all of them:

1. Click to select a job
2. Shift+click to select a range
3. Right-click for bulk actions

Only actions valid for all selected jobs will be available.
