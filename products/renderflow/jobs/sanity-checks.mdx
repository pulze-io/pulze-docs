Sanity checks validate your scene before submission to catch common issues that would cause render failures. Currently available for **3ds Max** only, with more applications coming soon.

## Managing checks

Access the **Sanity Check** page from the sidebar to:

- Enable or disable individual checks
- Edit check parameters
- Create custom checks
- Test checks against your current scene

## Creating custom checks

For advanced users, you can create your own sanity checks using MaxScript. Your script must return `true` (check passed) or `false` (check failed).

## Open source

The sanity check catalog is open source. Feel free to contribute new checks or improvements at [github.com/pulze-io/renderflow-sanity-check](https://github.com/pulze-io/renderflow-sanity-check).

## General checks

| Check | Level | Default | Description |
|-------|-------|---------|-------------|
| Untitled filename | Critical | Enabled | Scene must be saved (not untitled) |
| Autoback filename | Warning | Disabled | Filename should not contain "autoback" |
| Recover filename | Warning | Disabled | Filename should not contain "recover" |
| Empty scene | Critical | Enabled | Scene must contain objects |
| Render type | Warning | Enabled | Render type should be set to "View" |
| Production rendering | Critical | Enabled | Render mode must be set to production |
| Pixel aspect | Warning | Disabled | Pixel aspect ratio should be 1.0 |
| Active camera | Warning | Enabled | Scene should have an active camera |
| Single frame | Warning | Disabled | Time output should be single frame |
| Frame range | Warning | Disabled | Time output should be range |
| Frame step | Warning | Enabled | Nth frame should be set to 1 |
| Frame rate | Warning | Disabled | Frame rate should match expected value |
| Output path | Warning | Enabled | Output path must be configured |
| Output movie format | Critical | Enabled | Output cannot be AVI or MOV |
| Network path | Warning | Disabled | Output should be a network path |
| Display unit type | Critical | Disabled | Scene display units should match expected |
| System unit type | Critical | Disabled | System units should match expected |
| Render elements count | Warning | Enabled | Scene should have render elements |
| Render elements active | Warning | Enabled | Render elements should be enabled |
| Objects without materials | Warning | Enabled | All objects should have materials |
