The Statistics page provides comprehensive insights into your render farm's performance, resource utilization, and job processing efficiency. Access it from the **Chart** icon in the left sidebar.

<Frame caption="Statistics page overview showing multiple charts">
  <img src="https://placehold.co/900x500?text=Statistics+Overview" alt="Statistics overview" />
</Frame>

## Overview

Statistics are automatically collected and calculated by RenderFlow. The page displays nine different visualizations that help you understand:

- How efficiently your farm is being utilized
- Which nodes are doing the most work
- What software and render engines are being used
- How jobs are distributed across users
- Current job status breakdown

## Node utilization timeline

The timeline shows when each node was rendering and for how long.

<Frame caption="Node utilization timeline chart">
  <img src="https://placehold.co/900x400?text=Utilization+Timeline" alt="Node utilization timeline" />
</Frame>

| Feature | Description |
|---------|-------------|
| **Time Range** | Select from 1 hour to 7 days, or set a custom range |
| **Aggregation** | Data grouped by hour |
| **Filtering** | Show/hide specific nodes |
| **Sorting** | Sort by node name or total duration |

Each bar represents the time spent rendering during that period. Hover over a bar to see the exact duration and utilization percentage.

**Use this to:** Identify peak rendering times, spot underutilized nodes, and track rendering activity over time.
