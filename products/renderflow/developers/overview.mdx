## Quick start

<Steps>
  <Step title="Generate an API key">
    Go to **Settings → API Keys** in the RenderFlow UI.
  </Step>
  <Step title="Install an SDK or use the REST API directly">
    <CodeGroup>
    ```bash npm
    npm install @pulze-io/renderflow
    ```
    ```bash pip
    pip install pulze-renderflow
    ```
    </CodeGroup>
  </Step>
  <Step title="Connect and use">
    <CodeGroup>
    ```typescript JavaScript SDK
    import { RenderFlow } from "@pulze-io/renderflow";

    const rf = new RenderFlow({ apiKey: "your-api-key" });
    const jobs = await rf.jobs.list();
    ```
    ```python Python SDK
    from renderflow import RenderFlow

    rf = RenderFlow(api_key="your-api-key")
    jobs = rf.jobs.list()
    ```
    </CodeGroup>
  </Step>
</Steps>

Or use the [REST API](/products/renderflow/developers/rest-api) directly, the [CLI](/products/renderflow/developers/cli) for scripting, or the [MCP server](/products/renderflow/developers/mcp) for AI-powered control.
