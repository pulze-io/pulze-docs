### Key concepts

- **API keys** — Used to authenticate all API requests.
- **Render jobs** — The core unit of work in RenderFlow. Submit, monitor, and manage jobs via the API.
- **Webhooks** — Receive real-time notifications when render jobs change status.
- **Rate limits** — API requests are rate-limited to ensure fair usage across all users.
