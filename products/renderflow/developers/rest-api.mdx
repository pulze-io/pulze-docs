RenderFlow provides a REST API for managing jobs, nodes, tasks, pools, and more. Interactive API documentation is also available at `http://localhost:44442/docs` on any machine running RenderFlow.

<Frame caption="Interactive API documentation (Swagger UI)">
  <img src="https://placehold.co/900x500?text=API+Documentation+Interface" alt="API documentation" />
</Frame>

## Base URL

```
http://<ip>:44442/api/v1
```

Replace `<ip>` with the IP address of any machine running RenderFlow. Use `localhost` for your local machine.

## Authentication

All requests require an API key in the `x-renderflow-api-key` header:

```
x-renderflow-api-key: your-api-key-here
```

Generate keys from **Settings → API Keys**. See the [Authentication](/products/renderflow/developers/authentication) page for details.

### Create a job

<div style={{ display: 'flex', alignItems: 'center', gap: '8px', marginBottom: '12px' }}>
  <span style={{ background: '#eab308', color: 'white', padding: '2px 8px', borderRadius: '4px', fontSize: '13px', fontWeight: 600 }}>POST</span>
  <code>/jobs/create</code>
</div>

Creates a new render job.

**Request body**

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `name` | string | Yes | Job name |
| `file` | string | Yes | Path to the scene file |
| `type` | TaskType | Yes | Job type (e.g., `3dsmax.render`, `blender.render`, `cinema4d.render`) |
| `status` | string | Yes | Initial status: `pending` or `suspended` |
| `host` | object | Yes | Host application — `{ id, name, version }` |
| `engine` | object | No | Render engine — `{ id, name, version }` |
| `plugins` | array | No | Required plugins — `[{ id, name, version }]` |
| `resolution` | string | No | Output resolution (e.g., `1920x1080`) |
| `output` | array | No | Output paths — `[{ type, enabled, path }]` |
| `frame` | string | No | Frame range (e.g., `1-100`) |
| `pool_id` | string | No | Target pool ID |
| `priority` | number | No | Priority (0-100, default: 25) |
| `dependencies` | string[] | No | Job IDs this job depends on |
| `nodes` | string[] | No | Whitelisted node IDs |

<CodeGroup>
```bash curl
curl -X POST "http://localhost:44442/api/v1/jobs/create" \
  -H "x-renderflow-api-key: your-api-key-here" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "My Render Job",
    "file": "S:\\projects\\scene.max",
    "type": "3dsmax.render",
    "status": "pending",
    "host": { "id": "3dsmax", "name": "3ds Max", "version": "2025" },
    "engine": { "id": "vray", "name": "V-Ray", "version": "6.0" },
    "frame": "1-100",
    "priority": 50
  }'
```

```python Python
import requests

response = requests.post(
    "http://localhost:44442/api/v1/jobs/create",
    headers={
        "x-renderflow-api-key": "your-api-key-here",
        "Content-Type": "application/json"
    },
    json={
        "name": "My Render Job",
        "file": "S:\\projects\\scene.max",
        "type": "3dsmax.render",
        "status": "pending",
        "host": {"id": "3dsmax", "name": "3ds Max", "version": "2025"},
        "engine": {"id": "vray", "name": "V-Ray", "version": "6.0"},
        "frame": "1-100",
        "priority": 50
    }
)

job = response.json()
print(f"Created: {job['name']} ({job['_id']})")
```
</CodeGroup>

### Update a job

<div style={{ display: 'flex', alignItems: 'center', gap: '8px', marginBottom: '12px' }}>

|-------|------|-------------|

```bash curl
curl -X PATCH "http://localhost:44442/api/v1/jobs/60f7b1a2c3d4e5f6a7b8c9d0" \
  -H "x-renderflow-api-key: your-api-key-here" \
  -H "Content-Type: application/json" \
  -d '{"priority": 80, "limit": 5}'
```

```python Python
import requests

job_id = "60f7b1a2c3d4e5f6a7b8c9d0"

requests.patch(
    f"http://localhost:44442/api/v1/jobs/{job_id}",
    headers={"x-renderflow-api-key": "your-api-key-here"},

)
```

```python Python

response = requests.get(
    "http://localhost:44442/api/v1/jobs/events",
```

## Use cases

<AccordionGroup>
  <Accordion title="Slack notifications">
    Send a Slack message when jobs complete or fail:

    ```python
    import requests
    import time

    API_URL = "http://localhost:44442/api/v1"
    API_KEY = "your-api-key-here"
    SLACK_WEBHOOK = "https://hooks.slack.com/services/..."

    def check_jobs():
        headers = {"x-renderflow-api-key": API_KEY}
        jobs = requests.get(f"{API_URL}/jobs", headers=headers).json()

        for job in jobs:
            if job["status"] in ["completed", "failed"]:
                notify_slack(job)

    def notify_slack(job):
        emoji = ":white_check_mark:" if job["status"] == "completed" else ":x:"
        requests.post(SLACK_WEBHOOK, json={
            "text": f"{emoji} Job *{job['name']}* {job['status']}"
        })

    while True:
        check_jobs()
        time.sleep(300)
    ```
  </Accordion>
  <Accordion title="Daily farm report">
    Generate a daily summary of farm activity:

    ```python
    import requests
    from datetime import datetime

    API_URL = "http://localhost:44442/api/v1"
    HEADERS = {"x-renderflow-api-key": "your-api-key-here"}

    def generate_report():
        jobs = requests.get(f"{API_URL}/jobs", headers=HEADERS).json()
        nodes = requests.get(f"{API_URL}/nodes", headers=HEADERS).json()

        completed = len([j for j in jobs if j["status"] == "completed"])
        failed = len([j for j in jobs if j["status"] == "failed"])
        working = len([j for j in jobs if j["status"] == "working"])

        idle_nodes = len([n for n in nodes if n["status"] == "idle"])
        busy_nodes = len([n for n in nodes if n["status"] == "busy"])
        offline_nodes = len([n for n in nodes if n["status"] == "offline"])

        return f"""
        RenderFlow Daily Report - {datetime.now().strftime("%Y-%m-%d")}
        Jobs: {completed} completed, {failed} failed, {working} in progress
        Nodes: {idle_nodes} idle, {busy_nodes} busy, {offline_nodes} offline
        """
    ```
  </Accordion>
  <Accordion title="Auto-priority for urgent projects">
    Boost priority for jobs with specific tags:

    ```python
    import requests

    API_URL = "http://localhost:44442/api/v1"
    HEADERS = {"x-renderflow-api-key": "your-api-key-here"}
    URGENT_TAGS = ["urgent", "deadline", "client_review"]

    def check_and_boost_priority():
        jobs = requests.get(f"{API_URL}/jobs", headers=HEADERS).json()

        for job in jobs:
            if job["status"] == "pending" and job["priority"] < 80:
                job_tags = job.get("tags", [])
                if any(tag in URGENT_TAGS for tag in job_tags):
                    requests.patch(
                        f"{API_URL}/jobs/{job['_id']}",
                        headers=HEADERS,
                        json={"priority": 90}
                    )
                    print(f"Boosted priority for: {job['name']}")
    ```
  </Accordion>
  <Accordion title="Node health monitor">
    Alert when nodes go offline unexpectedly:

    ```python
    import requests
    import time

    API_URL = "http://localhost:44442/api/v1"
    HEADERS = {"x-renderflow-api-key": "your-api-key-here"}
    known_offline = set()

    def monitor_nodes():
        nodes = requests.get(f"{API_URL}/nodes", headers=HEADERS).json()
        for node in nodes:
            if node["status"] == "offline":
                if node["_id"] not in known_offline:
                    known_offline.add(node["_id"])
                    print(f"ALERT: Node went offline: {node['name']}")
            else:
                known_offline.discard(node["_id"])

    while True:
        monitor_nodes()
        time.sleep(60)
    ```
  </Accordion>
  <Accordion title="Automatic job cleanup">
    Archive completed jobs older than 7 days:

    ```python
    import requests
    from datetime import datetime, timedelta

    API_URL = "http://localhost:44442/api/v1"
    HEADERS = {"x-renderflow-api-key": "your-api-key-here"}

    def cleanup_old_jobs(days=7):
        jobs = requests.get(f"{API_URL}/jobs", headers=HEADERS).json()
        cutoff = datetime.now() - timedelta(days=days)

        for job in jobs:
            if job["status"] == "completed":
                completed_at = datetime.fromtimestamp(job.get("updated_at", 0) / 1000)
                if completed_at < cutoff:
                    requests.post(f"{API_URL}/jobs/{job['_id']}/archive", headers=HEADERS)
                    print(f"Archived: {job['name']}")
    ```
