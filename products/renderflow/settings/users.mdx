---
title: "Users"
description: "User accounts in RenderFlow."
---

RenderFlow tracks users automatically when they connect. The Users page shows everyone who has connected to RenderFlow.

<Frame caption="Users settings page">
  <img src="https://placehold.co/900x400?text=Users+Settings" alt="Users settings" />
</Frame>

| Field | Description |
|-------|-------------|
| **Name** | Username from the computer (read-only) |
| **Alias** | Display name |
| **Group** | Assigned user group |
| **Role** | Job title or role |
| **Email** | Contact email |
| **Computer** | The node associated with this user |
