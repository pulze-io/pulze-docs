## General

| Setting | Description |
|---------|-------------|
| **Appearance** | Choose light or dark theme, configure auto-close behavior |
| **Account** | Sign in with your Pulze account for licensing |

## User management

These settings require administrator permissions.

| Setting | Description |
|---------|-------------|
| **Users** | View and edit user profiles (alias, email, role, group) |
| **Groups** | Create and manage user groups |
| **Policies** | Configure permissions for each group |

## Farm configuration

These settings require administrator permissions.

| Setting | Description |
|---------|-------------|
| **Pools** | Create node pools to organize your render farm |
| **Licenses** | Manage which Pulze licenses RenderFlow uses (server only) |
| **Repository** | Set the shared folder path accessible by all nodes |
| **Mapped Drives** | Configure network drives to mount before rendering |

## System

These settings require administrator permissions.

| Setting | Description |
|---------|-------------|
| **Backup** | View backup history and create manual backups (server only) |
| **API Keys** | Generate keys for external API access |
| **System** | Maintenance mode and automatic idle detection |
