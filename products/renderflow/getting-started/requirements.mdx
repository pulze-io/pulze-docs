## Hardware

|  | Minimum | Recommended |
|--|---------|-------------|
| **CPU** | 2 cores | 4 cores |
| **RAM** | 8 GB | 16 GB |
| **HDD** | 64 GB | 128 GB |

## Operating system

- Windows 10 or Windows 11
- Admin privileges required during installation
- Shared network drive accessible from all workstations and render nodes

## Folders & registry

RenderFlow uses the following locations:

| Location | Purpose |
|----------|---------|
| `%PROGRAMFILES%\Pulze\RenderFlow` | Installation directory |
| `C:\ProgramData\RenderFlow` | Database (server mode), logs, and config files |
| `%APPDATA%\RenderFlow` | UI cache and additional config data |

**Registry entries:**
- `HKLM\Software\Microsoft\Windows\CurrentVersion\Uninstall\RenderFlow` — Add/Remove Programs entry
- `HKLM\Software\Microsoft\Windows\CurrentVersion\Run\RenderFlow` — Auto-start on login

**Required permissions:**
- Read/Write access to `C:\ProgramData\RenderFlow`

## Firewall

### Ports

| Port | Protocol | Purpose |
|------|----------|---------|
| **44442** | TCP | General API, log and event streaming, integrations |
| **44443** | UDP | Server discovery broadcasting |
| **44444** | TCP | Database access (server) |
| **44445** | TCP | Supervisor service |

### Executables

All located in `%PROGRAMFILES%\Pulze\RenderFlow`:

| Executable | Purpose |
|------------|---------|
| `RenderFlow.exe` | User interface |
| `rfsv.exe` | Service component |
| `resources\rfpm.exe` | Performance Monitor |
| `resources\mongod.exe` | MongoDB database |
| `resources\mongosh.exe` | MongoDB shell |
| `resources\mongodump.exe` | MongoDB backup |
| `resources\mongorestore.exe` | MongoDB restore |

## Internet access

The server requires internet access for authentication, licensing, and updates. Whitelist the following addresses:

| Domain | Purpose |
|--------|---------|
| `*.pulze.io` | Licensing, authentication, updates |
| `*.renderflow.com` | Mobile app service |
