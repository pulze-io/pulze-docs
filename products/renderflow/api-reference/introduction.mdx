## RenderFlow API

The RenderFlow API lets you programmatically manage render jobs, monitor progress, and integrate RenderFlow into your pipeline.

### Base URL

```
https://api.pulze.com/renderflow/v1
```

### Authentication

All API requests require an API key. Include it in the `Authorization` header:

```
Authorization: Bearer YOUR_API_KEY
```

See the [authentication guide](/products/renderflow/developers/authentication) for details on obtaining and managing API keys.
